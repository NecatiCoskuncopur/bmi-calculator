
# BMI Calculator

## ➡️ [Live Demo](https://bmi-calculator-appnc.netlify.app//)

## :computer: Setup

1. Clone the repository and install its features.

```
yarn install
```

2. Use the project locally (localhost).

```
yarn start
```

## :fire: Technologies and Libraries
